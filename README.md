# tp_integradorOE

Proyecto desarrollado como Trabajo Práctico Integrador para la materia de Ingeniería de Sistemas.

El objetivo del proyecto es automatizar el registro de solicitudes de soporte mediante un chatbot de Telegram desarrollado en Python e integrado con SQL Server.


Python 3
- Telegram Bot API
- python-telegram-bot
- SQL Server
- pyodbc
- python-dotenv
- Bizagi Modeler
- Visual Studio Code
